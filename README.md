# GiftLink Sparkle

GiftLink Sparkle is a clean, whimsical full-stack capstone scaffold aligned to the Skills Network **GiftLink** course structure.

It keeps the required folders:
- `giftlink-backend`
- `giftlink-frontend`
- `.github/workflows/main.yml`

It also aligns to the clearly visible course requirements from your uploaded notes:
- MongoDB connection in `giftlink-backend/models/db.js`
- gift endpoints in `giftlink-backend/routes/giftRoutes.js`
- route registration in `giftlink-backend/app.js`
- search and auth routes
- React pages for main, search, details, login, register, profile, and user listings
- GitHub Actions CI/CD lint + build workflow
- optional Docker/Kubernetes deployment samples

## Whimsical palette
No pinks are used. The visual theme uses:
- twilight navy
- periwinkle
- sky blue
- mint
- butter yellow
- porcelain white
- mossy green accents
